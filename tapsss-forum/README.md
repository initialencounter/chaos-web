# 联萌社区 · Web 论坛浏览器

给大家介绍一个小工具——**联萌社区 Web 版论坛浏览器**。

不想打开 App？没有安卓设备？在浏览器里也能刷帖子、看录像、查战绩了。

## 访问地址

> [联萌社区](http://tapsss.initenc.cn/)

---

## 主要功能

### 帖子浏览

- 首页支持按「最新」「热门」「关注」三种模式切换
- 支持上滑加载更多，置顶帖独立展示
- 帖子详情页包含完整正文、评论及楼中楼回复

### 搜索

- 全局关键词搜索帖子

### 录像回放

在帖子详情页可以直接在线观看录像，目前支持：

- 扫雷
- 数字华容道
- 舒尔特方格

### 用户主页

点击任意用户头像即可进入其主页，可查看：

- 基本信息（昵称、等级、排名）
- 多游戏战绩生涯，涵盖扫雷、拼图、数独、2048、非数独、舒尔特表格

---

## 技术说明

纯前端 Web 应用，基于 Vue 3 + TypeScript + Vite 构建，数据实时来自游戏服务器，本站不存储任何用户数据。
